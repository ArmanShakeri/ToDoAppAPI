ToDo Application A RESTful ToDo API


Arman Shakeri

Description
This is a simple ToDo API with FastAPI.

Methods

  • Get list of all existing task
  • Get an existing task
  • Create new task
  • Delete a task by id
  • Update an existing task

Parameters

  • Required: id={id} - An integer that serves as the unique identifier for the task.
  • Required: title={title} - A string that contains the title of the task.
  • Required: description={description} - A string that contains a more detailed description of the task.
  • Required: done={done} - A boolean that indicates whether the task has been completed.
